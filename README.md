# Containerized Flask Application

A Python Flask application containerized with Docker and deployed on Amazon ECS using Amazon ECR.

##  AWS Services Used

- Amazon ECS
- Amazon ECR
- AWS Fargate
- IAM
- VPC
- Security Groups

##  Features

- Dockerized Flask application
- Container image stored in Amazon ECR
- Deployed on Amazon ECS (Fargate)
- Publicly accessible using ECS Public IP
- Secure networking using Security Groups

##  Tech Stack

- Python
- Flask
- Docker
- AWS ECS
- Amazon ECR

##  Output

The Flask application is successfully deployed as a Docker container on Amazon ECS and accessed through the ECS Task Public IP.
